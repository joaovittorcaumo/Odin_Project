To create a list where the paragraph doesn't matter, you can use the ul, and each item within ul, must be created using li. Example:

  • ITEM 1
  • ITEM 2
  • ITEM 3

To create an ordered list, you must use te ol element, and again, each element within the list must be inbetween the li element. Example:

  1. ITEM 1
  2. ITEM 2
  3. ITEM 3
