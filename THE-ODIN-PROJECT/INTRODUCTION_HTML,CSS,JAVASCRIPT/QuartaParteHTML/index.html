ABSOLUTE AND RELATIVE LINKS

Absolute links are used to link your page to other page's through out the internet. Example: click me

Relative links are used to link with other pages in the same website. Example: About. This works because the files are in the same directory, but for more advanced projects, we want to have the index page at the root directory and the other ones in the child directory

To link the index to another file in a different directory, you must include the path of the file. Example: About

IMAGES

To display an image in html, we use < img > element. Unlike other elements, img doesn't use the closing parameter. Instead of wrapping the content, it embeds an image into the page using a src attributo wich tells the browser where the image is located.
